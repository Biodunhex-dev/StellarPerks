# StellarPerks

StellarPerks is a blockchain-based loyalty rewards platform built on the Stellar network. It enables businesses to issue tokenized loyalty points and allows users to earn, transfer, and redeem rewards securely using Stellar assets.

## Overview

Traditional loyalty systems are often fragmented and difficult to integrate across platforms. StellarPerks provides a unified solution by leveraging Stellar’s fast, low-cost infrastructure to create a transparent and efficient rewards ecosystem.

The platform allows businesses to create custom loyalty tokens while users interact with those tokens through secure wallet-based transactions.

## Features

- Tokenized loyalty points using Stellar assets  
- User wallet generation and management  
- Trustline setup for asset authorization  
- Reward issuance and redemption  
- Balance tracking and transaction visibility  
- Fast and low-cost transactions on Stellar Testnet  

## Architecture

The project consists of three main components:

- Backend: Node.js with Express for API services  
- Blockchain Layer: Stellar SDK for transaction handling  
- Frontend: React application for user interaction
